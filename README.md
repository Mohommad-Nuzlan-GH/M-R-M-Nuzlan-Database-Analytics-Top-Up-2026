# Databases and Analytics Assignment - NorthStar Urban Mobility and Logistics

This repository contains the notebooks and datasets for the **Databases and Analytics** assignment, focused on analysing and solving operational issues at NorthStar Urban Mobility and Logistics.

## Branch Structure

- **Main branch**: Contains the analysis notebooks using SQL in R, R analytics, Python data processing, MongoDB development, and query optimisation.
- **Second branch**: Includes the 9 cleaned datasets used for analysis.

## Notebooks

1. **1_Data_Cleaning_NS.ipynb**: Data cleaning and preparation.
2. **2_SQL_in_R_NS.ipynb**: SQL queries in R for operational analysis.
3. **3_R_Analytics_NS.ipynb**: Visualisation and KPI-based analytics in R.
4. **4_Python_Analysis_NS.ipynb**: Deep Anlaysis performed using python.
5. **5_MongoDB_Development_NS.ipynb**: MongoDB document design and data insertion & Indexing and query performance improvement in MongoDB.

## Datasets

## How to Run the Notebooks

1. Clone the repository.
2. Open the notebooks in **Google Colab**.
3. Upload the cleaned datasets when prompted.
4. Follow the instructions in each notebook.

## Technologies Used

- Python, R, SQL, MongoDB Atlas, Google Colab
